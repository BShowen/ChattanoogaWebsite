jQuery methods

prev()

  • prev() This is a DOM traversal method. You use it to select the preceding sibling of each element in a list of elements. You can optionally pass in a string argument and the preceding sibling will only be selected if it matches the argument. For example, you can select all of the <label> elements that precede the <input> elements with this code.$("input").prev();You can select all of the elements that preced the <input> elements only if they're <span> elemtents with this code. $("input").prev("span")

find()

  • find() This DOM traversal method will find and return all the matching elements in a set. You pass in a string argument which is the element you are searching for. For example, you can select all the <p> elements and then find all the <span> elements within the <p > elements with this code$("p").find("span");

attr()

  • attr() There are two versions of this method. This method either sets or gets an element's attribute. The getter version accepts one argument which will return the attribute value. The setter version will set the attribute to a specified value. To get an attribute you pass in the name of the attribute for which you want the value of. To set an attribute you pass in the first argument as the attribute name and the second argument as the attribute value.

css()

  • css() There are two versions of this method. This method either sets or gets an element's css. The getter version accepts one argument which will return the css property value. The setter version will set a css property to a specified value. To get a css property value you pass in the name of the css property for which you want the value of. To set a css property value you pass in the first argument as the css property name and the second argument as the css property value.

addClass()

  • addClass() Use this method to set the class name of an element or set of elements. You pass in a string argument which will become the class name(s) of the element(s). If you wish to add multiple classes to an element then you pass in a a single string with space separated class names. For example, lets set the class name of the <h1> element to "center bold" with this code$("h1").addClass("center bold");

removeClass()

  • removeClass() This method will remove a class or classes from and element or set of elements. You pass in a string argument which is the name of the class or classes you wish to remove from the element. Lets remove the "center" and "bold" classes from the <h1> element with this code$("h1").removeClass("center bold");

toggleClass()

  • toggleClass()This is a very easy way to add/remove a class from an element. You pass in a string argument which is the class name to be added or removed from the element. This method will check the element and if the element already has the class name, it will remove it. If the element doesn't have the class name, it will add it. Simple!

html()

  • html() There are two versions of this method. When you use this method without any arguments it will return the HTML content for the selected element. When you pass in an argument it will set the HTML content of an element to the HTML string argument that you passed into the method. For example, lets set the HTML content of the body to say "Hello jQuery" with this code.$("body").html("<h1> Hello jQuery!</h1>");

hide()

  • hide() This method makes it very easy to hide an element. Simply call this method on the selected element and it will be hidden from view. Lets hide all the <h1> elements with this code.$("h1").hide();

show()

  • show()This method makes it easy to show an element if its hidden. Simply call this method on the selected element and it will be shown on the webpage. Lets show all the hidden <h1> elements on the webpage with this code.$("h1.hidden").show();

each()

  • each() This method will execute a function once for each element in an array. You pass in the function to be executed as the only parameter. Say we want to change the text color of all the <h1> elements using the each() method. We would use this code$("h1").each(function(){ $(this).css("color", "blue") });there is a much easier way to do this though. Most jQuery methods that return a set of jQuery objects will automatically iterate through the set for you. We can do the same exact thing with this code.$("h1").css("color", "blue");

jQuery event methods

animate()

  • The animate() methodThe jQuery animate method works by allowing you to specify new CSS properties and values for the selected element and apply those properties and values over a specified amount of time. The syntax for this method looks like this animate({cssProperty: cssValue}, milliseconds, callbackFunction). When you use a callback function it will be run after the CSS properties have been applied by the first parameter of the method. This is useful when you want to perform an action only after the CSS properties have been fully applied to the selected element.

delay()

  • delay()This method is used to delay the animation or effect of the selected element. For example, lets show a div quickly and the hide the same div slowly. $("#div1").delay("fast").fadeIn();$("#div1").delay("slow").fadeOut();

stop()

  • stop()This method is used to stop the current animation of an element. When you pass in a boolean parameter of "true" then all of the animations in the queue for this element are removed. When a user clicks an animated element repeatedly, the animation will play multiple times in succession. This is undesirable and we use this method to prevent that behavior. We can use stop(true) to stop the current animation and then start a new one. If the user continues to click then the animation will be stopped and restarted with each click. This results in the animation playing only one time, no matter how many times a user clicks or activates an animated element.

finish()

  • finish()This method works similar to the stop() method. The finish() method will stop all currently-running animations, clear the animation queue, and complete any animations for the selected element.

ready()

  • ready() This event method is jQeury's equivalent of the window.onload() function in JavaScript. Any code that depends on the DOM should be places inside the ready() function. This is because the ready() function will be called and run only when the document is ready.

click()

  • click() This event method is applied to an element and the code inside the click() function will be executed when the element is clicked. This method accepts one argument as a function to be called. For example$("#submit_btn").click(function(){ //code to be executed });

mouseover()

  • mouseover() This event method is applied to an element and it accepts one argument as a function to be executed when the user moves their mouse over the element.

hover()

  • hover() This event method works similarly to the mouseover() method. However, the hover() method accepts two arguments as functions to be called. The first argument is executed when the mouse enters the element and the second argument is executed when the mouse exits the element.

jQuery methods to work with forms

val()

  • val() This jQuery method is used to get or set the value of a form control. To get the value of a form control you first need to select the form control with a jQuery selector and then call the val() method on the selected element. To set the value of a form control you first need to select the form control with a jQuery selector and then call the val() method on the selected element and also pass in a string argument to the val() method like this$("#name").val("new value");This method will return all the matching elements, not just the first matching element. So if you want to change all of the <input> elements you can use this code$("input").val("new value");However, if you want to change the value of a single element then you can select the specific element by its index. For example, to change the first <input> element you can use this code $("input").eq(0).val("new value"); or to change the third <input> you can use$("input").eq(2).text("new value");Understand that you access the specific element by its index in the return value of the jQuery selector.

text()

  • text() This jQuery method is used to get or set the value of a form field or form control. To get the value of an element you first select the element and then call the text() method on the selected element. To set the value of the element you select an element and call text() on the element and pass in a string argument as the new value to be used. This method will return all of the matching elements, not just the first matching element. So if you want to change all of the <h1> elements you can use this code$("h1").text("new value");However, if you want to change the value of a single element then you can select the specific element by its index. For example, to change the first <h1> element you can use this code$("h1").eq(0).text("new value");or to change the third <h1> you can use$("h1").eq(2).text("new value");Understand that you access the specific element by its index in the set that is returned by the jQuery selector.

next()

  • next() This jQuery method is use to get the next node, or next sibling, of an element. This is useful because the specific element you want to select may not have an id or class name associated with it which can make it difficult to select with a jQuery selector. What you can do is select the parent element and then call next() to select the next element, which would be the element you are trying to select. For example, you can select the <span> element that comes directly after the input field with id "first_name" with this code$("#first_name").next();

submit()

  • submit() This jQuery method is used to submit the corresponding form to the server for processing. You will usually use this method within the method that validates form data. If the form data is valid then you call the submit() method, otherwise you display proper error messages to the user.

focus()

  • focus() This jQuery method is used to highlight a form control or form field. You can only use this method on one form control element at a time.
